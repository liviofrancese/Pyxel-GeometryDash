### Pour définir le y d'un obstacle:
- La valeur par défaut de l'obstacle est y_min (tout le plus en bas possible). Si vous voulez mettre l'obstacle plus en haut, il faut juste mettre: -y ==> y étant le nombre de pyxel a monté
<br></br>
- Pour faire tourner un spike, il faut rajouter le paramètre: "turned": true
- Pour faire une fake orb (orb non fonctionnel), il faut rajouter le paramètre: "used": true
